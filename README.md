# Tiza

Data visualization library, based in D3.js.

License: GPLv3

## Features

### yearlyContractsBar
Recieves year summary data, split by role, and generates three yearly count and amount bar graphs, one for each role.

### contractTypePie
Recieves type summary data, split by role, and generates three type amount pie charts, one for each role.


### moneyFlow
Recieves relation summary data, i.e., a collection of nodes and links, and represents a network.
